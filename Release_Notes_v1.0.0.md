# Foundation Zero v1.0.0

Foundation Zero has ceased being continuously redesigned and is now a stable platform for engineering discipline.

## Scope
Version 1.0.0 provides a strictly evidence-driven governance system, separating knowledge from decisions, defining the promotion pipeline for operational capabilities (Skills and Agents), and providing onboarding (BOOTSTRAP) and certification protocols.

## Known Limitations
- The system is untested across multiple independent downstream projects.
- Mathematical verification currently only checks structural integrity, not semantic value.

## Migration Notes
- Downstream projects should import **Foundation Zero Lite** (Constitution, Doctrine, and 3 playbooks) rather than cloning the entire monorepo.
- Treat Foundation Zero as a compiler/dependency, not an application.

## Breaking Changes
- Proactive generation of Skills and Agents is permanently banned. Capabilities must now be earned through evidence.
- "Evidence Debt" has been replaced with a strict `Evidence Status` in proposals.
- Narrative success claims are banned; only raw execution proofs are accepted.

## Future Intentionally Deferred
- Conflict Resolution Protocol
- Doctrine Challenge Protocol
- Hardcoded Growth Budgets
- Formal Service Hierarchies

*No further governance changes are proposed until at least three independent business implementation projects have been completed using Foundation Zero.*
