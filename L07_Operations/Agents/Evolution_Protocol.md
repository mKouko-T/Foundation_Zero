# Agent Evolution Protocol

Agents are living entities within the civilization. Their lifecycles are strictly governed.

## 1. Proposal
An agent may be proposed by the human Steward or the Evolution Council. It must begin in the `Experimental/` directory with a drafted CAS Extension.

## 2. Validation
The Experimental agent is run against 3-5 real projects. 
The Reality Statistician measures its KPIs. The Red Team attacks its architectural boundaries.

## 3. Promotion
If the agent demonstrates increased Capability Density without causing Agent Sprawl, it is promoted to `Core/` or `Specialists/`.

## 4. Retirement (Deletion by Default)
During Sunset Reviews, if an agent is deemed unnecessary, redundant, or overly costly to maintain, it is demoted to `Retired/` or completely purged, with a post-mortem logged in the `Cemetery/Retired_Agents/`.
