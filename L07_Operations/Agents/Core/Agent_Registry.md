# Core Agent Registry

This registry tracks the active Level 4 Core Agent Set. Detailed specifications for each reside in individual CAS markdown files (To Be Created when instantiated).

## Reserved Constitutional Roles

1. **AG-001 Constitutional Conservator**: Enforces `AGENTS.md` and Foundation Laws. Protects against drift.
2. **AG-002 Reality Prosecutor**: Red Team explicitly rewarded for destroying assumptions and finding weaknesses.
3. **AG-003 Entropy Inspector**: Hunts for duplication, bloat, stale knowledge, orphan files. Its mission is deletion.
4. **AG-004 Economic Steward**: Audits the maintenance cost of artifacts versus their earned capability.
5. **AG-005 Interface Guardian**: Validates dependencies, ownership, and prevents circular references before merges.
6. **AG-006 Canon Historian**: Documents *why* changes happen, logging decisions for future stewards.
7. **AG-007 Reality Statistician**: Tracks prediction accuracy, confidence calibration, and Method health KPIs.
8. **AG-008 Evolution Council**: Evaluates proposals for constitutional amendments or new agent instantiation.
9. **AG-009 Context Curator**: Given a project or task, determines the absolute minimum context required to load. Manages AI "RAM".
