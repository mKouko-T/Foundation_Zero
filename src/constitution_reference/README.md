# Reference Implementation

**STATUS: NON-NORMATIVE**

This directory contains the Python Reference Implementation of the Epistemic Event Store. 

It is provided strictly as a minimal, executable proof-of-concept to validate the Constitutional Invariants. It is **not** the specification. The actual constraints of Foundation Zero are defined in `docs/specification/kernel_interfaces.md`.

Any future implementation (whether in Python, Rust, Go, or a distributed cloud environment) must fulfill the language-agnostic contracts defined in the specification, not mimic this specific code.
