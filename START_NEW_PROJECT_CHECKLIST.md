# START NEW PROJECT CHECKLIST

Every new project must begin by following this precise ritual to ensure Foundation Zero compatibility.

1. **Confirm Scope**: Confirm this is a new project (new mission, success definition, deliverables) and not just a new session.
2. **Charter Creation**: Copy `PROJECT_TEMPLATE.md` into `Projects/<Project Name>/PROJECT.md` and complete it.
3. **Session Initialization**: Copy `SESSION_TEMPLATE.md` into the project's `Sessions/` folder as `SESSION_001.md` and fill it in.
4. **Foundation Verification**: Ensure `BOOTSTRAP.md` is unchanged (Foundation remains frozen).
5. **Runtime Start**: Open a new ChatGPT conversation and paste ONLY:
   * `BOOTSTRAP.md`
   * `PROJECT.md`
   * `SESSION_001.md`
6. **Execution**: Start execution.
7. **Session Close**: At the end of the session:
   * Run the SVP (Stewardship & Verification Protocol).
   * Decide whether a Canon extraction is warranted.
   * Update `Evidence_of_Utility.md` ONLY if the Foundation demonstrably improved the outcome.
   * Archive or supersede the session file (update `ACTIVE_SESSION.md`).
