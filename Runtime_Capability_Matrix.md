# RUNTIME CAPABILITY MATRIX

**Purpose:** To prevent "wrong runtime syndrome." Whenever a task begins, the Foundation immediately knows which runtime should perform it. Neither runtime owns the truth. Foundation owns the truth.

| Task | ChatGPT (Cloud) | Antigravity (Local) |
| :--- | :--- | :--- |
| **Deep reasoning & Strategy** | ✅ Primary | Support |
| **Repository cleanup & Hygiene** | Review | ✅ Primary |
| **Git & Version Control** | Review | ✅ Primary |
| **Research synthesis & Theology** | ✅ Primary | Support |
| **Local automation & Terminal** | No | ✅ Primary |
| **Odoo implementation planning** | ✅ (Joint) | ✅ (Joint) |
| **Writing documentation** | ✅ Primary | Save / Organize |
| **Large-scale refactoring** | Review | ✅ Primary |
| **Constitutional review** | ✅ Primary | Support |
| **Local database migration** | Review | ✅ Primary |
| **Agent / Subagent Orchestration** | No | ✅ Primary |

## Runtime Posture
*   **ChatGPT**: Think. Challenge. Research. Compress. Audit. Teach. Red Team. Synthesize. Build methodologies.
*   **Antigravity**: Execute. Read/Write repository. Git. Automation. Code. Testing. Building. Deploying. Running workflows.
