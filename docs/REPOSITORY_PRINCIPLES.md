# Repository Principles

1. **Canonical before generated.**
2. **Generated artifacts are disposable.**
3. **Evidence is immutable.**
4. **State is compiled, never handwritten.**
5. **Specification precedes implementation.**
6. **Determinism over convenience.**
7. **Small commits over large migrations.**
8. **Repository stability outweighs elegance.**
9. **Governance scales better than memory.**
10. **Knowledge is the product; tooling is the factory.**
