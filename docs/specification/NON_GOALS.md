# Non-Goals

To prevent decades of drift, Foundation Zero is explicitly **NOT**:

- A Graph Database (We use event ledgers; graphs are disposable projections).
- A Note-Taking System (We extract knowledge from systems like Obsidian; we do not replace them).
- An AI Agent Framework (LangGraph, etc., are external applications that invoke our Kernel API).
- A Vector Database or RAG Pipeline (Embeddings are application-level indexes, not kernel state).
- An Ontology (The Kernel is agnostic to the domain ontology).
- A Workflow Engine.
- A File Storage System.
