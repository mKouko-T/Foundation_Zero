# Complexity Budgets

Foundation Zero operates on explicit budgets. Every architectural category has a finite limit.

## Principle
Budgets must be justified. Increasing a budget requires an Architectural Decision Record (ADR) detailing the economic trade-off (Maintenance Burden vs Benefit).

When a budget is reached, **something must be deleted before a new item can be added**.

## Categories
While specific numbers are determined by the Steward and ADRs, budgets exist for:
- Kernel Interfaces
- Ontology Roots
- Permanent Playbooks
- Permanent Skills
- Governance Documents

The default response to reaching a budget is **subtraction**, not expansion.
